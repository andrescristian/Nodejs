Sintaxe básica do JavaScript

; no final NÃO É OBRIGATÓRIO
- Conceito de Sentenças (Linha de código)
- Conceiro de Bloco (Formado por chaves{})
- SyntaxError --> Erro de digitação

Variáveis
- No JavaScript as Variáveis são dinamicamente tipadas(O tipo depende do dado atribuído)
- let  (Cria uma Variável no JS)
- const(Cria uma Constante no JS)

Tipos de Variáveis/Constantes
1º Numéricas    Reais e Inteiros)
2º String       String de caracteres
3º Booleano     Mostra se um valor é Verdadeiro ou Falso(true/false)

Comandos
- console.log      Exibe na tela
- console.clear    Limpa tela
- typeof           Mostra o "Tipo de Variável"(String, Numérico, ...)

- Regras para definir o "Nome da Variável"(Identificador):
1º Não usar espaço
2º Não usar acentos e caracteres especiais
3º Não usar letras minúsculas
4º Definir nomes que estejam relacionados ao problema (código limpo)
5º Usar a técnica do Camelo(Camel Case) para nomes de variáveis compostos
    Exemplo: nome cliente --> NomeCliente (Camel Case)

Operadores Aritméticos
- São usados em Operações Aritméticas junto às Constantes ou Variáveis
